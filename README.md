# DBA Portfolio

신입 Database Administrator(DBA)를 목표로 진행하는 실무 중심 포트폴리오입니다.

이 프로젝트에서는 MySQL을 기반으로 데이터베이스 구축, 운영, 성능 튜닝, 백업 및 복구, Replication, Migration 등을 직접 실습하며 기록합니다.

---

## 🎯 Project Goal

- MySQL 환경 구축 및 운영
- 데이터베이스 설계(ERD)
- SQL 작성 및 데이터 관리
- 성능 분석 및 튜닝
- 트랜잭션(Transaction)
- 백업 및 복구(Backup & Recovery)
- Replication(이중화)
- 데이터 마이그레이션(Migration)
- Linux 환경에서 DB 운영

---
